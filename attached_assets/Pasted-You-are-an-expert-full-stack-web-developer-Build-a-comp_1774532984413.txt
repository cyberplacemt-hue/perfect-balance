You are an expert full-stack web developer. Build a complete, 
production-ready e-commerce website for a horseshoe supplier.

=== PROJECT OVERVIEW ===
Business: Online store selling professional horseshoes and 
horseshoe nails (B2B + B2C)
Brand name: "Perfect Balance"
Target: Farriers, equestrian clubs, private horse owners
Language: Russian
Stack: Node.js + Express backend, HTML/CSS/JS frontend
Single deployable project on Replit

=== SERVER (server.js) ===
- Express.js server serving static files from /public
- REST API endpoints:
  GET  /api/products       — return all products as JSON
  GET  /api/products/:id   — single product
  POST /api/order          — save order (log to console + return confirmation)
  POST /api/contact        — handle contact form
- Products data hardcoded in server.js as array (no database needed)
- CORS enabled
- Port: process.env.PORT || 3000

=== PAGE: catalog.html ===
- Sidebar filters:
  Category (All | Horseshoes | Nails)
  Sort (Price asc/desc | Name | Popular)
  Search input with live filtering
- Products grid (3 cols desktop, 2 tablet, 1 mobile)
- Fetches from GET /api/products
- Renders cards dynamically with JS
- "В корзину" button on each card
- Empty state if no results
- Skeleton loading placeholders while fetching

=== PAGE: product.html ===
- Reads ?id= from URL
- Fetches from GET /api/products/:id
- Layout:
  Left: product image (large) with zoom on hover
  Right: name, sizes, specs table, price (pair + box),
  quantity selector, "В корзину" button,
  "Заказать оптом" link to contact
- Specs accordion (expandable)
- Related products (3 cards at bottom)

=== PAGE: cart.html ===
- Reads cart from localStorage
- Table: product | size | qty (editable) | price | remove
- Order summary block (right):
  Subtotal, shipping note, total
  "Оформить заказ" button → checkout.html
- Empty cart state with illustration and CTA

=== PAGE: checkout.html ===
- Multi-step form (3 steps with progress bar):
  Step 1: Contact info (name, phone, email)
  Step 2: Delivery (city, address, transport company)
  Step 3: Confirmation (order review + submit)
- POST to /api/order on final submit
- Order summary sidebar (sticky on desktop)
- Success page after submission
